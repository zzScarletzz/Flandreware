to be or not to be


これは教育向けに作成して、公開しております。
悪意のあるように使用しないでください。
これは研究用に作られたプログラムです。
私のプログラムを改変して、自作したなど言うなど、販売などに使用しないでください。

このプログラムは仮想マシンで行ってください。

必要なゲーム
th06
 => 東方紅魔郷　～ the Embodiment of Scarlet Devil.
ゲーム内で、エクストラステージでスコアを稼いでください。
